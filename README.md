Interactive Cybersecurity Defense Quiz

This project is a single-page web application (SPA) designed to test and educate users on fundamental cybersecurity vulnerabilities and defenses through scenario-based questioning.

It was developed as part of an Internal Assessment (IA) for a Computer Science course to demonstrate interactive simulation and web development skills using plain HTML, JavaScript, and Tailwind CSS.

 Features

Scenario-Based Questions: Four common cyber attack scenarios (Brute-Force/Rate Limiting, Malware, Cross-Site Scripting, SQL Injection).

Interactive Hints: Users can request hints if they get stuck, promoting active learning.

Fixed Scoring: Each correct answer awards 25 points, tracking the total score out of 100.

Responsive Design: Optimized for viewing on both desktop and mobile devices.

Technology Stack

HTML: Structure

JavaScript (JS): Quiz logic, state management, scoring, and interaction.

Tailwind CSS (via CDN): Modern, utility-first styling for aesthetics and responsiveness.
